# PiedPiper Landing Page

Лендинг-страница для проекта PiedPiper - децентрализованной интернет-платформы с криптовалютой PiedPiperCoin.

## Описание проекта

PiedPiper представляет собой автономную peer-to-peer сеть с распределенным хранилищем данных, использующую универсальное сжатие, ускоренное распределение задач и end-to-end шифрование. Проект создает полностью децентрализованный интернет, где пользователи предоставляют собственные ресурсы хранения, вычислений и сети.

## Особенности CSS

- Фиксированная позиция (position: fixed)
    
- Фон: rgba(0, 119, 101, 1)
    
- Логотип слева, меню справа
    
- Float-based layout
    

## Секции

- Центрирование контента
    
- Адаптивные отступы
    
- Цветовая схема: бирюзовый, белый, темные оттенки
    

## Типографика

- Размеры шрифтов от 18px до 72px
    
- Использование em для курсива
    
- Жирный текст через класс .bold-text
    

## Команда

- Flexbox для размещения карточек
    
- Круглые фотографии членов команды
    
- Центрированные подписи

## Навигация

Проект использует:

- Отступы и размеры шрифтов в относительных единицах
    


## Запуск проекта

1. Убедитесь, что все файлы находятся в одной директории
    
2. Откройте файл `index.html` в браузере
    
3. Страница готова к просмотру
